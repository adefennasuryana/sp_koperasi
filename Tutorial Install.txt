Tutorial Install Source code POS Kasir Berbasis Website

1. Download dan Install XAMPP di komputer anda
2. Ekstrak file sp_koperasi.zip yang sudah di berikan dengan Winrar atau sejenisnya
3. Copy folder posv1, lalu paste ke folder htdocs 
4. Aktifkan Apache2 dan MySQL pada XAMPP di komputer anda
5. Buka browser anda dan buka url http://localhost/phpmyadmin 
6. Buat database baru dengan nama posv1
7. Import database aplikasi di folder posv1/database sql/posv1.sql
8. Buka browser anda dan buka url http://localhost/sp_koperasi/
9. Login dengan username adminkasir dan password secret sebagai level admin
10. Selesai
